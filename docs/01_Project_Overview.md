# Project Overview: Highground Beta

## Executive Summary
**Highground Beta** (Internal Code: *Shal Phyoke Admin*) is a comprehensive enterprise resource planning (ERP) and operational management system designed specifically for the modern food and beverage industry. It streamlines complex workflows—from inventory tracking and procurement to real-time order management and financial analytics—into a single, high-performance web interface.

## Business Objectives
- **Operational Efficiency:** Automate manual inventory tracking and reduce procurement errors.
- **Data-Driven Decisions:** Provide real-time financial and performance metrics via interactive dashboards.
- **Resource Optimization:** Minimize waste through intelligent threshold alerts and automated stock management.
- **Scalability:** Build a robust foundation capable of supporting multiple locations and expanding menu complexity.

## Target Users
- **Operations Managers:** For oversight of inventory, procurement, and financial health.
- **Floor Staff/Kitchen:** For real-time order processing and menu status updates.
- **Purchasing Officers:** For managing vendor relationships and streamlining the market list process.
- **Business Owners:** For high-level performance tracking and strategic planning.

## Key Stakeholders
- **Highground Leadership:** Project sponsors and strategic directors.
- **Shal Phyoke Operations Team:** Primary users and domain experts.
- **Development Team:** Full-stack engineers and UI/UX designers.

## Technology Stack
- **Frontend:** React 19, Vite, TanStack Router
- **Styling:** Tailwind CSS v4, DaisyUI
- **State Management:** Zustand
- **Backend/Database:** Supabase (PostgreSQL)
- **AI Integration:** Google Gemini GenAI (for automated data processing)
- **Visualization:** Recharts
- **Validation:** Zod & React Hook Form

## Development Timeline
- **Phase 1: Foundation (Q4 2025):** Database schema design, core authentication, and inventory module.
- **Phase 2: Operations (Q1 2026):** Order management, procurement workflows, and real-time synchronization.
- **Phase 3: Intelligence (Q2 2026):** Financial analytics, AI-assisted data processing, and dashboard implementation.
- **Phase 4: Beta Launch (March 2026):** Current phase—optimization, documentation, and pilot testing.
